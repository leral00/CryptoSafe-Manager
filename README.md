# CryptoSafe-Manager
Описание проекта
CryptoSafe Manager — это приложение на Python для безопасного хранения учетных данных (логинов и паролей).
Текущая версия представляет собой архитектурный фундамент будущего менеджера паролей.

Цель работы
Целью данной работы является разработка архитектурной основы настольного приложения для безопасного хранения учетных данных с использованием принципов:
1) модульности и масштабируемости
2) архитектурного паттерна MVC
3) изоляции криптографической логики
4) работы с локальной базой данных SQLite
5) подготовки проекта к дальнейшему расширению и внедрению реальных механизмов защиты данных

На текущем этапе приложение позволяет:
1) Создавать записи 
2) Отображать список записей в таблице
3) Изменять существующие записи
4) Удалять записи
5) Сохранять данные в локальной базе SQLite 
6) Создавать резервную копию базы данных
7) Работать через графический интерфейс 
8) Автоматически создавать базу данных при первом запуске
9) Запускать автоматические тесты через GitHub Actions

Реализованный этап разработки:
Sprint 1 — Архитектурный фундамент
На текущем этапе реализованы:
1) Структура проекта с разделением по слоям
2) Архитектура MVC 
3) Подключение базы данных SQLite
4) Создание базы vault.db
5) Базовый графический интерфейс на Tkinter
6) Заглушка криптографического слоя
